# 第13回課題 p5.js 学生作品展示

`index.html` をWebサーバー経由で開くと、提出作品を10件ずつ一覧表示できます。

## 開き方（Windows）

このフォルダで `start_viewer.ps1` を右クリックして「PowerShellで実行」するか、PowerShellで次を実行します。

```powershell
.\start_viewer.ps1
```

ブラウザで <http://localhost:8000> を開いてください。停止するときはPowerShellで Ctrl+C を押します。

## 補足

- 学籍番号・氏名で絞り込めます。
- 「拡大 ↗」で作品だけを別タブ表示できます。
- 各作品はコードから推定した再生時間後に自動で先頭から繰り返します。
- 提出物のうちp5.js本体のみだったファイルは、プレビュー不可として表示します。
- 各作品は個別のiframe内で実行され、表示中の10作品が同時に動きます。
